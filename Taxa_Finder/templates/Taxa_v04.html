Taxa Finder

Taxa Finder uses the ncbi taxonomy database to find the path from your query to the root of the tree of life.
It can process one or several queries. If several queries are input, please write them on one line and seperate them by commas ",".








Short Lineage: Displaying a shorter lineage instead of the full, where entries flagged as suppressed in the GenBank entry lineage, are hidden.

Find Last Common Node: If more than one query is entered and this checkbox is ticked, the Taxa Finder finds you the last common taxonomic node.
